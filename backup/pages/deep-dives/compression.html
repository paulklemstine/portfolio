Compression Theory & Information Limits

The mathematical impossibility of universal compression — and what the pigeonhole principle really means.

The Fundamental Impossibility

No injective map exists from {0,1}ⁿ → {0,1}ⁿ⁻¹

By the pigeonhole principle, any map from 2ⁿ strings to 2ⁿ⁻¹ strings must have collisions. This proves that no lossless compression algorithm can shrink every input.

Key Results

KRAFT Kraft Inequality

For prefix-free codes: Σ 2^(-ℓᵢ) ≤ 1. The fundamental constraint on code lengths.

SHAN Shannon Entropy

Shannon entropy ≥ 0. No source can have negative information content.

SRC Source Coding Theorem

Both achievability and converse: optimal compression rate equals entropy. Proven in both directions.

DPI Data Processing Inequality

Processing cannot increase information. A fundamental constraint on any computation.

INC Incompressible Majority

Fraction of k-bit-incompressible strings > 1 − 2⁻ᵏ. Most strings resist compression.

FUTILE Recompression Futile

Repeated compression cannot help. Compressing the output of a compressor gains nothing.
