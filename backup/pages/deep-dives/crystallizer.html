Deep Dive #7

The Crystallizer

Neural network weights that crystallize onto the integer lattice — with provable stability, safety, and robustness guarantees.

Crystallization Loss

L(w) = sin²(πw) = 0  ⟺  w ∈ ℤ

The crystallization loss function has zeros precisely at integers. Training minimizes this loss alongside the task loss, driving weights toward integer values. The loss is π-periodic and symmetric about integers.

Crystallization Dynamics

Provable Safety Guarantees

LYA Lyapunov Stability

Crystallization loss ≥ 0 always, and loss = 0 ⟺ equilibrium. The system is Lyapunov stable.

GRAD No Gradient Explosion

Gradient explosion is mathematically impossible in crystallized networks. The spectral radius is exactly 1.

LIP 1-Lipschitz Robustness

Crystallized layers are 1-Lipschitz: small input perturbations cause at most proportional output changes.

RELU Rational Exactness

ReLU preserves ℚ: integer weights + rational inputs = exact forward pass. No floating-point errors.

The Pendulum Analogy

Crystallization dynamics are isomorphic to a physical pendulum system. Each weight behaves like a pendulum attracted to the nearest integer, with sin²(πw) as potential energy. This gives physical intuition for convergence.
