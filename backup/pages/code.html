Source Code

All source code for the factoring engines, ECDLP solvers, C extensions, and visualization scripts.

Key Source Files

siqs_engine.py

Self-Initializing Quadratic Sieve -- primary factoring workhorse (48-72 digits). Multi-worker sieve, K-S multiplier, DLP combining, GF(2) Gauss elimination.

Factoring

gnfs_engine.py

General Number Field Sieve -- all 5 phases: polynomial selection, factor base, lattice sieve, SLP/DLP combining, SGE, Hensel sqrt. Targets 40+ digits.

Factoring

resonance_v7.py

Unified dispatcher -- heterogeneous factoring driver. Routes to SIQS, GNFS, or ECM based on digit size and factor balance. Pythagorean tree Path 1 + Spectral Compass.

Driver

cfrac_engine.py

Continued Fraction factoring engine -- Pythagorean tree climb-up as CF expansion. L(1/2) complexity. 40d in 1s, 45d in 57s.

Factoring

siqs_core_c.c

C extension for SIQS sieve core -- int16 sieve array, batch trial division, 35x faster than Python. The 95% hotpath.

C Extension

siqs_poly_c.c

C extension for SIQS polynomial generation -- Gray code switching, incremental offset updates, batch polynomial evaluation.

C Extension

gnfs_sieve_c.c

C sieve + verify extension for GNFS -- __int128 arithmetic, Miller-Rabin primality, int64 fast path for rational norms.

C Extension

gnfs_lattice_sieve_c.c

C lattice sieve for GNFS -- special-q lattice enumeration, 43-50x speedup over line sieve. Enables 50+ digit GNFS.

C Extension

block_lanczos_v2.c

Block Lanczos algorithm for GF(2) linear algebra -- O(n^2*w/64) complexity. Essential for GNFS matrices above 15K rows.

C Extension

ec_kangaroo_shared.c

Shared-memory multi-process kangaroo for ECDLP (secp256k1) -- van Oorschot-Wiener parallelism, lock-free DP table via mmap, Levy flight jumps, 128-bit positions. 2.2-3.1x with 6 workers.

C Extension
