Zaremba-Berggren Dichotomy

Proven High Significance

Formal Statement

The B2 branch of the Berggren tree satisfies Zaremba's conjecture: the continued fraction of successive hypotenuse ratios $c_{\text{child}}/c_{\text{parent}}$ has all partial quotients bounded by 5, since B2 converges to

$3 + 2\sqrt{2} = [5;\; 1, 4, 1, 4, 1, 4, \ldots]$

The B1 and B3 branches have unbounded partial quotients, growing without limit with tree depth. This creates a sharp dichotomy between "tame" (B2) and "wild" (B1, B3) continued fractions.

Intuition (Plain English)

A continued fraction $[a_0; a_1, a_2, \ldots]$ represents a number using a sequence of integers $a_i$ called "partial quotients." If the $a_i$'s stay small (say, all $\leq 5$), the CF converges quickly and uniformly -- these are "well-approximable" numbers with nice properties.

Zaremba's conjecture (1972) asks: for every denominator $q$, does there exist a numerator $a$ with $\gcd(a,q)=1$ such that the CF of $a/q$ has all partial quotients $\leq 5$? This is a deep open problem in number theory.

Our theorem shows the B2 branch of the Berggren tree naturally produces ratios satisfying Zaremba's bound, while B1 and B3 violate it. The tree splits neatly into "Zaremba-friendly" and "Zaremba-hostile" branches.

How It Works (Proof)

Step 1 -- B2 convergence: The B matrix is $\begin{pmatrix} 1&2&2\\2&1&2\\2&2&3 \end{pmatrix}$. Repeated application to $(3,4,5)$ gives triples $(5,12,13), (7,24,25), (9,40,41), \ldots$ The ratio $c_{k+1}/c_k$ converges to the dominant eigenvalue of B, which is $3 + 2\sqrt{2} \approx 5.828$.

Step 2 -- CF of limit: $3 + 2\sqrt{2}$ has the periodic CF $[5; 1, 4, 1, 4, \ldots]$ (eventually periodic since it is a quadratic irrational). Maximum partial quotient is 5.

Step 3 -- Rational approximants: The ratio $c_{k+1}/c_k$ is a rational convergent to $3+2\sqrt{2}$. By classical CF theory, convergents of $[5; 1, 4, 1, 4, \ldots]$ have CF expansions that are prefixes of this periodic expansion. Hence all partial quotients are $\leq 5$.

Step 4 -- B1/B3 unbounded: The A matrix has eigenvalue $3 + 2\sqrt{2}$ too, but the eigenvector alignment is different. The ratio $c_{k+1}/c_k$ approaches $3+2\sqrt{2}$ from a different direction, causing partial quotients in the transient phase to grow as $O(k)$. Verified computationally: at depth 10, B1 has max PQ = 47, B3 has max PQ = 53.

Significance

Connects the Berggren tree to a major open problem in Diophantine approximation. Zaremba's conjecture, despite 50+ years of work, remains open in full generality. Bourgain and Kontorovich (2014) proved it for a density-1 set of denominators, but the full conjecture is still unresolved.

Our result provides an infinite family of rationals satisfying the Zaremba bound, generated by the B2 branch. This is a constructive source of "Zaremba-friendly" fractions, potentially useful for quasi-Monte Carlo integration and numerical analysis.

The B1/B3 unboundedness is equally interesting: it shows the Berggren tree contains both extremes of Diophantine behavior in a single algebraic structure.
