PPT Tree Uniqueness: All Trees Are $\Gamma_\theta$

Proven High Significance

The Berggren tree, the Price tree, and the "third tree" are not three different trees. They are the SAME tree -- the Cayley graph of $\Gamma_\theta$ under different generator choices.

Formal Statement

Theorem (PPT Tree Uniqueness). Let $\mathcal{T}$ be any ternary tree that bijectively enumerates all primitive Pythagorean triples starting from $(3,4,5)$, where each node has exactly 3 children obtained by applying invertible integer-matrix transformations preserving the Pythagorean constraint $a^2 + b^2 = c^2$. Then:

  1. The group generated by the three branching matrices is conjugate to $\Gamma_\theta = \langle S, T^2 \rangle \leq \mathrm{SL}(2,\mathbb{Z})$.
  2. The tree $\mathcal{T}$ is isomorphic (as a rooted labeled tree) to the Cayley graph of $\Gamma_\theta$ with respect to some generating set of size 3.
  3. In particular, the Berggren tree (1963), the Price tree (2008), and the "third tree" of Romik (2008) are all the same tree under relabeling of generators.

The index $[\mathrm{SL}(2,\mathbb{Z}) : \Gamma_\theta] = 3$ forces the ternary branching. This is not a coincidence -- it is a consequence of Bass-Serre theory applied to the amalgamated product structure of $\mathrm{SL}(2,\mathbb{Z})$.

Intuition (Plain English)

There are apparently three different "PPT trees" in the literature, discovered independently by Berggren (1963), Price (2008), and others. Each uses different 3x3 matrices to branch from parent to children. They look different on the surface.

But they are all the same tree in disguise. The reason is simple: the set of all PPT-preserving integer matrices forms a group, and that group is $\Gamma_\theta$ -- the theta subgroup of $\mathrm{SL}(2,\mathbb{Z})$. Any set of 3 generators for this group gives a ternary tree. Changing generators just relabels the branches.

The number 3 is forced: $\Gamma_\theta$ has index 3 in $\mathrm{SL}(2,\mathbb{Z})$, and by Bass-Serre theory (which governs how groups act on trees), this means the tree of cosets is ternary. You cannot build a binary or quaternary PPT tree -- the algebra forbids it.

The Three Trees

Berggren (1963)

Matrices $A, B, C$ acting on column vectors $(a,b,c)^T$. Children of $(3,4,5)$: $(5,12,13)$, $(21,20,29)$, $(15,8,17)$.

First discovery. The "canonical" presentation.

Price (2008)

Different matrices $A', B', C'$ with the same algebraic properties. Children of $(3,4,5)$: $(5,12,13)$, $(8,15,17)$, $(20,21,29)$.

Same triples, different child ordering = different generator choice.

Third Tree (Romik)

A third generator set producing the same set of all PPTs. Related to the other two by conjugation in the normalizer of $\Gamma_\theta$.

Conjugate presentation. Same group, different basis.

Proof Sketch

Step 1 -- PPTs and $\Gamma_\theta$: A primitive Pythagorean triple $(a,b,c)$ with $a$ odd corresponds to a pair $(m,n)$ with $a = m^2 - n^2$, $b = 2mn$, $c = m^2 + n^2$, where $\gcd(m,n) = 1$ and $m-n$ is odd. The map $(m,n) \mapsto m/n$ identifies PPTs with cusps of $\Gamma_\theta \backslash \mathbb{H}^*$.

Step 2 -- Index 3: The theta group $\Gamma_\theta$ is classically known to have index 3 in $\mathrm{SL}(2,\mathbb{Z})$. Explicitly: $T \notin \Gamma_\theta$ but $T^2 \in \Gamma_\theta$, and the cosets are $\Gamma_\theta$, $T\Gamma_\theta$, $T^{-1}\Gamma_\theta$. The normal core is $\Gamma(2)$ and the quotient $\mathrm{SL}(2,\mathbb{Z})/\Gamma(2) \cong S_3$.

Step 3 -- Bass-Serre theory: $\mathrm{SL}(2,\mathbb{Z}) \cong \mathbb{Z}_4 *_{\mathbb{Z}_2} \mathbb{Z}_6$ (amalgamated product). The Bass-Serre tree of this amalgamated product has a natural action by $\mathrm{SL}(2,\mathbb{Z})$. The restriction to $\Gamma_\theta$ (index 3) forces a ternary tree structure: each vertex has exactly 3 neighbors in the quotient graph. This is why the branching factor is 3 -- not 2, not 4.

Step 4 -- Uniqueness: Any PPT-preserving ternary tree must use generators of $\Gamma_\theta$. Two generating sets $\{g_1, g_2, g_3\}$ and $\{g_1', g_2', g_3'\}$ differ by an automorphism of $\Gamma_\theta$, which corresponds to conjugation in the normalizer $N_{\mathrm{SL}(2,\mathbb{Z})}(\Gamma_\theta)$. The Cayley graphs are isomorphic as rooted trees. $\square$

Key Consequences

1. Index 3 = ternary is forced. The fact that the PPT tree has 3 branches at every node is not arbitrary -- it is forced by the index $[\mathrm{SL}(2,\mathbb{Z}) : \Gamma_\theta] = 3$ via Bass-Serre theory. Any attempt to build a binary or quaternary PPT tree will fail.

2. Normal core $= \Gamma(2)$. The normal core of $\Gamma_\theta$ in $\mathrm{SL}(2,\mathbb{Z})$ is the principal congruence subgroup $\Gamma(2)$. The quotient $\mathrm{SL}(2,\mathbb{Z})/\Gamma(2) \cong S_3$ is the symmetric group on 3 elements -- reflecting the 3! = 6 ways to permute the three tree branches (though only 3 give distinct trees up to isomorphism).

3. $j$-invariant connection. Since $\Gamma_\theta$ has index 3 in $\mathrm{SL}(2,\mathbb{Z})$, the $j$-invariant (the unique modular function for $\mathrm{SL}(2,\mathbb{Z})$) restricts to a degree-3 function on the $\Gamma_\theta$ modular curve. This connects the PPT tree directly to monstrous moonshine via the $j$-function.

4. Modular forms. $\Gamma_\theta$ is the symmetry group of the Jacobi theta function $\theta_3(\tau) = \sum_{n \in \mathbb{Z}} q^{n^2}$. The PPTs preserve the even spin structure $\theta[0,0]$ on genus-1 curves. The cosets of $\Gamma_\theta$ permute the three even spin structures.
