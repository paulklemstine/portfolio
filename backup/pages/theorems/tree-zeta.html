Tree Zeta Function: No Functional Equation

Proven High Significance
Theorem. The Berggren tree zeta function $\zeta_T(s) = \sum_{\text{PPT}} c^{-s}$ has abscissa of convergence $\sigma_c = 1$ and Hausdorff dimension $\approx 0.6232$. It possesses: (a) no Euler product, (b) no functional equation ($s \leftrightarrow 1-s$), (c) no critical line. It is an "arithmetic Dirichlet series without automorphic structure."

Intuition

The Riemann zeta function $\zeta(s) = \sum n^{-s}$ has beautiful properties: an Euler product over primes, a functional equation relating $\zeta(s)$ to $\zeta(1-s)$, and (conjecturally) all zeros on a critical line.

The tree zeta function has none of these. It sums over PPT hypotenuses instead of all integers, and this restriction breaks every structural property. The sum-of-two-squares indicator is not multiplicative, so there is no Euler product. Without multiplicativity, there is no automorphic form behind it, no functional equation, and no critical line. The Riemann Hypothesis concept simply does not apply.

Proof Sketch

Abscissa of convergence: By Lehmer's formula, the count of PPTs with $c \leq X$ is $N(X) \sim X/(2\pi)$. By partial summation, $\zeta_T(s)$ converges for $s > 1$ and diverges for $s \leq 1$, giving $\sigma_c = 1$.

No Euler product: The indicator function $\mathbf{1}_{\text{sum of 2 squares}}(n)$ is not multiplicative. For example, $5$ and $13$ are sums of two squares, but $65 = 5 \times 13$ has two representations. The count function $r_2(n)$ involves a convolution, not a product.

No functional equation: Tested numerically: define $\phi(s) = \zeta_T(s) / \zeta_T(2-s)$. If a functional equation existed, $\phi$ would be constant. The coefficient of variation of $\phi$ is 1.16 (far from 0), ruling out any $s \leftrightarrow 2-s$ symmetry.

Hausdorff dimension: The set of hypotenuse values has natural density 0 (sparse in the integers) but the "dimension" computed via box-counting on the tree gives $\dim_H \approx 0.6232$.

Significance

This definitively rules out using analytic continuation, functional equation techniques, or RH-type conjectures to study the distribution of Pythagorean triples. The tree zeta function lives outside the Langlands program -- it cannot be lifted to an L-function. This is a fundamental structural result about the arithmetic of the Berggren tree.
