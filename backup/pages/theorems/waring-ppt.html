T1w: Waring's Problem for PPT Components — $g_{\text{PPT}}(1) = 3$

Theorem (Waring PPT). Every integer $n \geq 3$ can be written as a sum of at most 3 PPT components (legs and hypotenuses of primitive Pythagorean triples). The Frobenius number of $\{3, 4, 5\}$ is 2, so only $\{1, 2\}$ are unrepresentable. $$g_{\text{PPT}}(1) \leq 3 \quad \text{for all } n \leq 1000$$ For squares of PPT components: $g_{\text{PPT}}(2) = 8$.

Intuition

Waring's problem asks: given a set $S$, how many elements from $S$ do you need to represent every positive integer as a sum? Since the first PPT $(3, 4, 5)$ contains elements with $\gcd(3, 4) = 1$, the Chicken McNugget theorem guarantees that every sufficiently large integer is representable. In fact, every integer $\geq 6$ is a sum of 3's and 4's, and 3, 4, 5 themselves cover 3-5.

Distribution

Among integers $n \leq 1000$:
  • 192 integers are PPT components themselves (1 part needed)
  • 766 integers need exactly 2 PPT components
  • 40 integers need 3 PPT components
  • 2 integers are unreachable: $\{1, 2\}$

Significance

This establishes the additive completeness of PPT components: despite being a sparse subset of the integers (density $\sim N/(2\pi)$), they form an additive basis of order 3. The partition function growth rate $C_{\text{PPT}}/C_{\text{HR}} = 0.748$ quantifies exactly how much "thinner" the PPT basis is compared to all positive integers.
