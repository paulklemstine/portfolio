Dual Number Linearization

Proven High Significance
Theorem. Replace integers with dual numbers $a + a'\varepsilon$ (where $\varepsilon^2 = 0$). Then the Pythagorean equation $(a+a'\varepsilon)^2 + (b+b'\varepsilon)^2 = (c+c'\varepsilon)^2$ splits into: $$\text{Real part: } a^2 + b^2 = c^2 \quad\text{(quadratic, hard)}$$ $$\text{Dual part: } aa' + bb' = cc' \quad\text{(linear, easy)}$$ The $\varepsilon$-shadow is a linear equation that captures the tangent structure of the Pythagorean surface.

Intuition

Dual numbers are like "numbers with memory." The number $a + a'\varepsilon$ remembers both its value ($a$) and its derivative ($a'$). When you square it, the $\varepsilon^2$ term vanishes, leaving only first-order information.

This means the nonlinear Pythagorean equation $a^2 + b^2 = c^2$ has a hidden linear twin: $aa' + bb' = cc'$. The linear equation describes the tangent plane to the Pythagorean surface at each triple. It is the "easy" shadow of the "hard" quadratic constraint.

Proof

Step 1: Let $\varepsilon^2 = 0$. Expand $(a + a'\varepsilon)^2 = a^2 + 2aa'\varepsilon$. Similarly for $b$ and $c$.

Step 2: The equation $(a+a'\varepsilon)^2 + (b+b'\varepsilon)^2 = (c+c'\varepsilon)^2$ becomes:

$(a^2 + b^2) + (2aa' + 2bb')\varepsilon = c^2 + 2cc'\varepsilon$

Step 3: Matching real and dual parts: $a^2 + b^2 = c^2$ and $aa' + bb' = cc'$.

Step 4: The dual equation $aa' + bb' = cc'$ is linear in the primed variables. Given any PPT $(a,b,c)$, the set of valid $(a', b', c')$ forms a plane in 3D. This is the tangent plane to the Pythagorean cone at $(a,b,c)$. QED.

Significance

This reveals that every quadratic Diophantine equation has a linear "shadow" in the dual numbers. Unfortunately, the linear shadow alone cannot solve factoring because it requires knowing the base triple $(a,b,c)$ first (Obstruction #1: circularity). But it provides a clean algebraic framework for understanding the tangent structure of the solution space.
